# History Atlas - Interactive World Population Timeline

An interactive historical world map showing population changes from 3000 BC to 2025 AD.

## Features

- **Interactive World Map** - Drag, zoom, and explore historical regions
- **Time Slider** - Drag the timeline to see population changes through history
- **Population Visualization** - Circle markers sized and colored by population
- **Historical Context** - Major civilizations and regions for each time period
- **Info Panel** - Real-time statistics for the selected time period
- **Keyboard Navigation** - Use arrow keys to navigate through time

## How to Use

1. Open `index.html` in a web browser
2. Drag the time slider at the bottom to move through history
3. Click on any population marker to see details about that region
4. Use arrow keys (← →) for finer time navigation
5. Zoom and drag the map to explore different regions

## Data

Historical population estimates for 15 major time periods:
- 3000 BC to 1 AD: Ancient civilizations
- 500 AD to 1500 AD: Medieval period
- 1700 AD to 1950 AD: Industrial era
- 2000 AD to 2025 AD: Modern era

Population figures are estimates based on historical sources. Accuracy increases with more recent time periods.

## Technology

- **Leaflet.js** - Interactive mapping
- **Pure HTML/CSS/JavaScript** - No build tools required
- **OpenStreetMap** - Base map tiles
